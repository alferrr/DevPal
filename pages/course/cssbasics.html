CSS Basics

CSS Course

Introduction

CSS (Cascading Style Sheets) is the language used to control the visual presentation of HTML. In this course, you'll learn how to style text, layout pages, add colors, create responsive designs, and add animations. CSS works alongside HTML and JavaScript to bring webpages to life.

CSS allows you to separate content (HTML) from presentation (CSS), making your code cleaner, easier to maintain, and scalable for larger projects.

CSS Syntax

CSS rules consist of selectors, properties, and values:

  • Selector: chooses which HTML element(s) to style
  • Property: the aspect of the element you want to change (color, font-size, margin, etc.)
  • Value: the setting for the property

Example:

p {
  color: blue;
  font-size: 16px;
  line-height: 1.5;
}

You can also group selectors to apply the same styles to multiple elements: 

h1, h2, h3 {
  font-family: Arial, sans-serif;
  color: #333;
}

Selectors

Selectors allow you to target specific elements or groups of elements:

  • Element Selector: p { ... }
  • Class Selector: .classname { ... }
  • ID Selector: #idname { ... }
  • Group Selector: h1, h2, h3 { ... }
  • Pseudo-classes: a:hover { ... } – for special states like hover, focus, active
  • Pseudo-elements: p::first-letter { ... } – style parts of elements
  • Attribute Selectors: input[type="text"] { ... }

Colors, Backgrounds & Borders

CSS colors, backgrounds, and borders add visual style and separation to content.

Colors

  • Named colors (red, blue, green)
  • Hex codes (#ff0000)
  • RGB (rgb(255,0,0))
  • RGBA for transparency (rgba(255,0,0,0.5))
  • HSL (hsl(0, 100%, 50%))
  • CSS variables for theming (--primary-color: #3498db;)

Backgrounds

  • background-color
  • background-image (url('image.jpg'))
  • background-size, background-repeat, background-position
  • linear-gradient, radial-gradient for smooth color transitions

Borders

  • border-width, border-style, border-color
  • border-radius for rounded corners
  • box-shadow for depth

Example

.card {
  background: linear-gradient(135deg, #f6f8fa, #e9ecef);
  border: 2px solid #ccc;
  border-radius: 12px;
  box-shadow: 0 4px 6px rgba(0,0,0,0.1);
  padding: 20px;
}

Layouts & Positioning

Layout is how content is arranged on a page. Understanding layout is fundamental for designing professional-looking websites.

Block vs Inline Elements

  • Block elements: take full width, start on a new line (div, p, h1-h6, section)
  • Inline elements: take only the width of their content, do not start on a new line (span, a, strong, em)
  • Inline-block: behaves like inline but accepts width/height (display: inline-block;)

Positioning

  • static: default positioning, flows naturally in the page
  • relative: moved relative to original position
  • absolute: positioned relative to nearest positioned ancestor
  • fixed: stays in viewport even when scrolling
  • sticky: acts like relative until scroll passes threshold, then fixed

Layout Techniques

  • Flexbox: one-dimensional layout, aligns items in rows or columns (display: flex;)
  • CSS Grid: two-dimensional layout, control rows & columns (display: grid;)
  • Floats: old technique to wrap text around elements (float: left/right;)
  • Positioning with percentages: relative widths & heights for responsive layouts

Deep Practice Examples

/* Flexbox Example */
.container {
  display: flex;
  flex-direction: row;
  justify-content: space-between;
  align-items: center;
}

/* Grid Example */
.grid {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 20px;
}

Typography & Text Styling

Typography determines how readable and visually appealing your text is. CSS gives full control over fonts, size, spacing, alignment, and decoration.

Fonts & Text

  • font-family: sets typeface (sans-serif, serif, monospace)
  • font-size: controls size (px, em, rem, %)
  • font-weight: controls boldness (normal, bold, 100-900)
  • line-height: controls vertical spacing between lines
  • text-align: left, right, center, justify
  • text-decoration: underline, overline, line-through, none

Advanced Text Styling

  • letter-spacing and word-spacing
  • text-transform: uppercase, lowercase, capitalize
  • text-shadow: subtle depth and visual effects
  • hyphens: control word breaking (hyphens: auto;)

Practical Example

p {
  font-family: 'Roboto', sans-serif;
  font-size: 1.2rem;
  line-height: 1.6;
  letter-spacing: 0.5px;
  text-align: justify;
  color: #333;
}

Responsive Design Basics

Responsive design ensures your website looks good on all screen sizes, from phones to desktops.

Media Queries

  • Use @media to apply styles based on viewport width or height
  • Mobile-first design: define small screen styles first, then scale up
  • Breakpoints: 576px, 768px, 992px, 1200px (common)

Flexible Units

  • Percentages for widths (width: 80%)
  • em/rem for font sizes relative to root or parent
  • vh/vw for viewport-based sizing

Fluid Layout Example

.container {
  width: 90%;
  max-width: 1200px;
  margin: 0 auto;
}

@media (max-width: 768px) {
  .container {
    width: 95%;
  }
}

Organizing Your CSS Code

Writing clean, maintainable CSS is just as important as learning the syntax.

Best Practices

  • Use external CSS files instead of inline styles
  • Group related styles together
  • Use meaningful class and ID names
  • Comment your sections clearly
  • Reuse classes instead of repeating rules
  • Keep CSS specificity low to avoid conflicts

Folder Structure Example

/css
  base.css       /* reset and basic styles */
  layout.css     /* grid, flexbox, positioning */
  typography.css /* fonts, headings, paragraphs */
  components.css /* buttons, cards, modals */
  utilities.css  /* helpers: margin, padding, colors */

Using CSS Variables

:root {
  --primary-color: #3498db;
  --secondary-color: #2ecc71;
  --font-base: 'Roboto', sans-serif;
}
body {
  font-family: var(--font-base);
  color: var(--primary-color);
}
